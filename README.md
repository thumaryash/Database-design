# Database Design

This repository contains the database design diagrams for the project.  
The diagrams represent the structure of the database including tables, attributes, and relationships between different entities.

## Description
The database design was created using database modeling software.  
The images in this repository show the Entity Relationship (ER) diagrams and schema structure used for designing the database.

## Contents
- Database schema diagrams
- Entity Relationship (ER) diagrams
- Table relationships and structure

## Tool Used
The database design was created using a database design tool and exported as images.

## Purpose
The purpose of this repository is to document and visualize the database structure so that developers can easily understand how the data is organized and how different tables are related.
